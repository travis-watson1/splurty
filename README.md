# Splurty

Splurty is a simple quote generator created using Ruby on Rails that displays motivational quotes.

## Installation

First, run bundle install:

```bash
bundle install
```

Second, create a database:

```bash
rake db:create
```
```bash
rake db:migrate
```

## Deployed App

https://splurty-travis-watson.herokuapp.com
